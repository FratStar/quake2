Quake PlayPen: Lemuel Wilson - Quake 2


=====================

********************************************************
Gameplay:

The game is rather simple reach the set "happylimit" to win the game.
Weapons that don't fire lead can still kill so be careful.
Point of the game is to be come happy enough.


*********************************************************

Changelog:

Happiness is now a stat.
Happiness is displayed in the top middle of the screen.
Every weapon that fires lead is now a water gun.
Characters bounce persistently.
Q-Tip does "happy" damage.
Q-Tip is a weapon but it uses the model of the blaster/
Grenades spawn "gib head" due to limitations with finding map position that's persistent.
Happylimit is a console variable that can be edited on the fly.
